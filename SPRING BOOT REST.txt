

@JsonProperty("NombreAtributoX")
private tipoDato nombreAtributoX; -> Para Al serializar el objeto Java a JSON, el atributo nombreAtributoX 
	se representará en el JSON con el nombre "NombreAtributoX".
	Al deserializar JSON a objeto Java, si el JSON contiene "NombreAtributoX", su valor se asignará al
	atributo nombreAtributoX.

MULTIPARTFILE 

Metodos de objetos que son de la clase multipartFile.
	
file.getOriginalFilename() -> Devuelve el nombre original del archivo tal como lo subió el cliente.
	Ejemplo: "foto.png".

file.getContentType() -> Devuelve el tipo MIME del archivo pe si el archivo es png, jpg, pdf, Word, etc.
	En si los posibles valores serian: "image/jpeg", "image/png", "image/gif", "audio/mpeg", 	"audio/wav", "audio/ogg", etc.

file.isEmpty() -> Devuelve true si el archivo está vacío (no se envió o tiene tamaño 0).

file.getSize() -> Devuelve el tamaño del archivo en bytes.

file.getBytes() -> Devuelve el archivo completo en un array de bytes (byte[]).
	Útil para guardarlo en una DB o procesarlo en memoria.

file.getInputStream() -> Devuelve un InputStream del contenido del archivo.
	Ideal para procesar archivos grandes sin cargarlos completos en memoria.
