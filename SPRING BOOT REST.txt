

~~~~~~~ Personalizar los mensajes de error (opcional) ~~~~~~~

Se crea un archivo properties donde se manejan los mensajes personalizados.

* En resourcers/messages.properties *
NombreAnotacion.NombreEntidadX.nombreAtributo=mensajePersonalizado -> Crea un
	mensaje personalizado para cierta anotacion.
	Las anotaciones pueden ser: NotEmpty, NotBlank, etc.

Inyectar los mensajes personalizados a las anotaciones.

* En clase principal *
@PropertySource("classpath:messages.properties") -> Para que Spring cargue un archivo de propiedades 	adicional llamado messages.properties el cual vive en el classpath (src/main/resources/).

* En entities/NombreEntidadX *
@NombreAnotacion(message = "{NombreAnotacion.NombreEntidadX.nombreAtributo}")
private tipoDato nombreAtributo; -> Para Utilizar el mensaje personalizado cuando no se cumpla la 
	restricci√≥n de la anotacion.

Posteriormente se debe reiniciar el servidor.
