


problemas ----------------------------------

Se tuvo un problema al momento de hacer un despligue automatico, al principio se tenia un application.yaml
pero al cambiarlo por su equivalente en application.properties, resulta que su codificación cambia de 
utf-8 a iso-8520-1 y al hacer el despliegue este solo acepta archivos con la codificación utf-8.
Solución: menú file -> settings -> apartado izq editor -> file encodings -> un poco hasta abajo en donde
dice: default encoding for properties files -> seleccionar utf-8 -> apply -> ok.
