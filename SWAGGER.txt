*luego luego dentro de components:*

  securitySchemes:
    BearerAuth:
      type: http
      scheme: bearer
      bearerFormat: JWT -> Declara un esquema de seguridad llamado BearerAuth, con esto se define el 	tipo de autenticación que usa la API.
	Visualmente en swagger monstrara un candado donde se deberá ingresar el toekn.
	Dentro de cada petición se deberá colocar algo como:
		      security:
        		- BearerAuth: []
	para indicar que ese endpoint necesita esta autenticación.
