Enviar petición a servicio soap usando restTemplate ----------------------

val headers = HttpHeaders().apply {
    contentType = MediaType.APPLICATION_XML
    accept = listOf(MediaType.APPLICATION_XML)
} -> Crear los encabezados para la petición.
	La primer linea Indica que el cuerpo de la petición se enviará en formato XML
	y la segunda linea Indica que esperas recibir una respuesta en formato XML.

val request = HttpEntity<String>(xml, headers) -> Crea el cuerpo completo de la 
	petición HTTP que se enviará con RestTemplate.
	La variable XML es un string que almacena el request de la petición.

val response = RestTemplate().postForEntity(url, request, String::class.java) -> Realiza la peiticon al
	servicio.
	Lo común es que el serivioc soap se le envie una petición post.

val json = XML.toJSONObject(responseIssue.body.toString()) -> Extrae el cuerpo de la respuesta, lo

val jsonObject = JSONObject(json.toString()) -> Convierte el objeto json a JSONObject.
	Cuando tenga este tipo de dato, podemos usar los métodos: getJSONObject, getJSONArray, put,
