MIGRAR TABLAS DE SQL A NO-SQL

Este escenario cubre mover tablas de MySQL a mongoDb.
Se debe tener la conexión desde navicat hacia la conexión que contiene las tablas de MySQL

*En navicat*
click der sobre las tablas que se desea exportar -> export wizard -> json file -> next -> seleccionar todas
o determinadas tablas -> next -> next -> next -> start -> antes de cerrar esa ventana se nos dira donde
fue almacenada esa collection.

*En mongoDb compass*
Ingresar a una conexión -> Ingresar a una db -> crear una collecion -> ingresar a esa collection -> add
data -> import json or csv file.
