CLASE FILES

Clase utilitaria en Java para trabajar con archivos y directorios.

tempFile = Files.createTempFile(nombreArchivo, nombreExtension); -> Crear un archivo temporal que puede ser 
	utilizado para almacenar datos de manera temporal durante la ejecución de un programa. 
	El archivo temporal se crea en el directorio predeterminado para archivos temporales del sistema. 

Files.deleteIfExists(tempFile); -> Elimina cierto archivo temporal solo si este existe.
	Arroja la exception 'DirectoryNotEmptyException' si el archivo es un directorio y no pudo
	eliminarse porque el directorio no está vacío.
	Arroja la exception 'IOException' si hubo un problema de escritura o lectura.

Files.createFile(Path path) → crea un archivo vacío en la ruta especificada.
	La extensión depende completamente del nombre que le des en el Path.

createDirectory(Path path) → crea un directorio en la ruta especificada.
