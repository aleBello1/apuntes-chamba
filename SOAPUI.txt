Técnicamente se puede usar otros clientes como postman para consumir una api soap, pero auqnue
parezca increíble hay api's soap que aunque sean consumidas correctamente (request valido,
url y credenciales correctas) no se obtendrá respuesta alguna.
Es por eso que es mejor que al intentar consumir una api soap, de plano usar este cliente. *****
