import redis -> Importa la librería oficial de Redis para Python.

connector = redis.Redis(
    host=REDIS_HOST,
    port=REDIS_PORT,
    db=n,
    decode_responses=True
) -> Establecer conexión con redis.
	- db=n -> Selecciona a que db nos conectaremos, de las 16 disponibles (del 0 al 15).
	- decode_responses=True -> Redis transmite todo como bytes, esta configuracion convierte 
		los datos automáticamente a strings para que Python no te muestre cosas raras.

connector.set(REDIS_KEY, value) -> Guarda un dato como clave valor.
	- json.dumps(data) -> Si el valor a guardar es un objeto o arreglo de objetos es 			necesario guardalos como un objeto jsonString o array jsonString porque entre los
		tipos de datos que maneaja redis el que mas se adecua a esto es string. 
		Redis no puede almacenar directamente estructuras avanzadas como: listas de Python,
		diccionarios de Python u objetos de Python.
