////////////////////////////////////GLOSARIO/////////////////////////////////////////

Aplicación: conjunto de objetos, campos y otras funciones para cubrir un proceso empresarial.

Objetos: son tablas en la base de datos de Salesforce que almacenan un tipo de información en particular.

Trailhead Playground: Entorno seguro donde puede practicar las habilidades que está adquiriendo antes de aplicarlas en su trabajo real.


//////////////////////////////

Agregar un campo a determinado objeto: En la parte sup derecha buscar el logo de settings -> 
	opción setup -> click apartado object manager -> click sobre determinado objeto -> 
	apartado izq field and relationships -> new -> seleccionar el tipo de dato del campo
	-> sig -> asginar el nombre del campo y su id -> next -> next -> save.

Comprobar la creación del campo nuevo para un objeto: En App Launcher (Iniciador de aplicación) 
	(Iniciador de 
	aplicación) -> buscar dreamhouse -> click sobre determinad objeto ->
	seleccionar un registro -> apartado details -> buscar el campo recién creado.
